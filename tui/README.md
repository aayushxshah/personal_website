# 🖥️ aayush.app — Personal Terminal Website

Welcome to my personal website!  
This project is a playful, terminal-style web experience where you can explore my work and personality through commands.

## ✨ Features

- Fake terminal with fun commands (`ls`, `cd`, `date`, etc.)
- Custom ASCII art banner
- Commands to view my education, experience, summary, or social links
- Sarcastic error messages (because what’s a terminal without them?)
- Fully customizable with a single config file
